# PrivEsc + ReversShell (Windows)

Este repositorio recoge configuraciones, scripts y recursos relacionados con la escalada de privilegios y el acceso remoto en sistemas Windows mediante reverse shells. Se centra en la preparación, ejecución y gestión de estos procesos dentro de entornos Windows, incluyendo el uso de herramientas y configuraciones específicas.

Se abordan aspectos como la elevación de privilegios, ejecución de comandos con mayores permisos y establecimiento de conexiones remotas, junto con su integración en distintos escenarios de administración o análisis de sistemas.

El contenido está orientado a usuarios con conocimientos en sistemas Windows y redes que busquen comprender y trabajar con este tipo de configuraciones de forma estructurada y educativa.
